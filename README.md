# Proyecto API ASP.NET Core

Este repositorio contiene una API desarrollada en **ASP.NET Core** con operaciones CRUD y consumo de datos externos.

## Características
- CRUD completo para entidades.
- Conexión a SQL Server.
- Consumo de API externa.
- Documentación con Swagger/OpenAPI.
